# AuroraSauce Software ホームページ

AuroraSauce Softwareの公式ホームページです。

## 技術スタック

- Next.js 14
- TypeScript
- Tailwind CSS

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

## ビルド

```bash
# 本番用ビルド
npm run build

# 本番サーバーの起動
npm start
```
